## About the Project
The landing page is designed with a focus on a high-tech "Glassmorphism" aesthetic, combining transparency, blur effects, and vibrant gradients.

### Key Features:
* **Glassmorphism UI**: Advanced use of `backdrop-filter`, inner shadows, and semi-transparent layers.
* **Pixel Perfect**: Precise implementation based on the original Figma mockup.
* **Fluid Typography**: Leveraging the CSS `clamp()` function for seamless font scaling across all device sizes.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
* **Custom Form Elements**: Styled checkboxes, custom select menus, and interactive hover states.

## Tech Stack
* **HTML5**: Semantic structure and modern accessibility standards.
* **CSS3**: Flexbox layout, CSS Variables, Media Queries, and Keyframe animations.
* **JavaScript (Vanilla)**: Clean, library-free code for smooth scrolling, active section highlighting in the header, and form handling.
* **Fonts**: Druk Wide Cyr (Headings) and VK Sans Display (Body text).
* **Graphics**: Scalable SVG icons and optimized raster images.
